## 🚀 How to Access & Run

1. **Development Server**:
   ```bash
   # Start Laravel server
   php artisan serve
   
   # Start Vite dev server (for hot reload)
   npm run dev
   ```
   Kemudian buka: `http://127.0.0.1:8000`

2. **Production Build**:
   ```bash
   npm run build    # Build optimized assets
   php artisan optimize  # Cache config & routes
   ```
